# Security Policy

## Supported Versions
We only provide security updates for the latest version of PeerChat.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| < Latest| :x:                |

## Reporting a Vulnerability
We take the security of PeerChat very seriously. If you find a security vulnerability, please **do not open a public issue**. Instead, follow these steps:

1. **Email us**: Send a detailed report to `mathiraja@mathi.live`.
2. **Include Details**: Describe the vulnerability, how to reproduce it, and the potential impact.
3. **Wait for Response**: We will acknowledge your report within 48 hours and provide a timeline for a fix.

We follow a coordinated disclosure policy. We ask that you give us a reasonable amount of time to fix the issue before making it public.

## Our Commitment
PeerChat is designed for privacy. We use industry-standard encryption (libsodium) and peer-to-peer protocols to ensure your data never touches a server. We are committed to patching any flaws that might compromise this mission.
